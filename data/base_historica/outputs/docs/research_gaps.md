# Pendências de pesquisa

Dos 500 eventos selecionados, 497 exigem pesquisa por fontes específicas. Antes de declarar a base pronta, cada P0 deve receber fonte de nível A ou justificativa explícita, conforme a seção 16 do guia. Também faltam observações de mercado verificadas, custos, relações adicionais e a camada brasileira para a maior parte dos eventos externos.
